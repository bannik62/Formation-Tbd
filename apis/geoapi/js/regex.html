quantificateurs:


+ => minimum 1 fois sans limite maximum
* => peu importe le nombre de fois, 0 compris
? => entre 0 et une fois maximum
{n} (n doit être positif) => se répète exactement n fois (exemple {3} => forcément 3 fois)
{n, m} (n et m doivent être positifs) => se répète entre n et m fois exactement (exemple {2, 4} => entre 2 et 4 fois)
/ caractere d échappement {n,} (n doit être positif) => se répète un minimum de n fois, sans limite (exemple {2, } => supérieur ou égal à 2)

du coup par exemple: 0? => pas de 0 ou une seule fois
